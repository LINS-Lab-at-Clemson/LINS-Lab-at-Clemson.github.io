abstract = """HydroCNHS is an open-source, semidistributed hydrological model for coupled natural–human systems. Its four application programming interfaces allow users to integrate agent-based representations of reservoirs, river diversions, transbasin conveyance, drainage, and other human decisions while preserving routing logic and water balance. A unified configuration file organizes hydrological and human-system inputs, and parallel computing supports model calibration. A case study in the northwestern United States demonstrates the package and its potential for water-resources planning, uncertainty analysis, and more complex agent designs."""
